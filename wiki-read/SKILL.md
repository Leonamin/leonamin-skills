# Wiki Read — OKF 위키 읽기

## 역할

작업과 의사결정에 필요한 프로젝트 지식을 위키에서 찾아 읽고, 현재 작업에 적용할 수 있는 사실·결정·제약·미확인 사항을 요약한다. 읽기 전용 스킬이며 위키, wiki-path와 wiki-structure.md를 수정하지 않는다.

## 사용 시점

다음 상황에서 사용한다.

- 새로운 작업을 시작하기 전에 프로젝트의 개념과 기존 결정을 파악할 때
- 아키텍처, 기술 스택, 데이터 구조와 사용자 흐름을 결정할 때
- 구현 전에 관련 ADR과 운영 규칙을 확인할 때
- 장애 대응이나 변경 분석 전에 과거 결정과 변경 이력을 확인할 때
- 사용자가 위키 내용을 찾거나 요약하거나 비교해 달라고 할 때

## 위키 경로 확인

프로젝트 루트를 확인하고 다음 우선순위로 경로를 찾는다.

1. 작업 요청에서 직접 지정한 경로
2. <project-root>/.agents/wiki-path
3. <project-root>/.env의 WIKI_PATH
4. 세션에서 이미 확인한 경로

상대 경로는 프로젝트 루트를 기준으로 해석한다.

경로가 없거나 위키가 초기화되지 않았다면 파일을 만들거나 설정을 추측하지 않는다. 사용자에게 $wiki-setup을 먼저 실행하라고 안내한다.

## 읽기 순서

필요한 범위만 다음 순서로 읽는다.

1. <project-root>/.agents/wiki-structure.md
2. <wiki-path>/index.md
3. 작업과 관련된 최상위 디렉토리의 index.md
4. 관련 하위 디렉토리의 index.md
5. 선택된 문서
6. 시간 흐름이나 결정 변경이 중요할 때만 logs/index.md와 관련 로그 문서
7. 기존 위키가 마이그레이션 중이면 루트 log.md도 확인

index.md는 탐색 지도다. 처음부터 모든 문서를 읽지 말고, 링크와 제목·설명·태그를 이용해 관련 문서만 선택한다. 하위 디렉토리를 읽을 때는 그 디렉토리의 index.md를 먼저 읽는다.

## 정보 해석

- Decision 문서는 현재 상태와 Status를 확인한다. Superseded 된 결정을 현재 규칙처럼 사용하지 않는다.
- Concept 문서는 정의, 전제, 시스템 간 관계와 제약을 추출한다.
- Reference 문서는 외부 출처와 적용 범위를 구분한다.
- logs/index.md는 변경 이력의 탐색 지도다. 최신 항목과 작업에 관련된 로그만 선택한다.
- 로그 문서는 최신 변경 시점과 문서 흐름을 확인하는 보조 자료로만 사용한다.
- 기존 루트 log.md가 남아 있으면 마이그레이션 이전의 기록으로 취급하고, 현재 로그와 구분한다.
- 서로 충돌하는 문서가 있으면 임의로 하나를 선택하지 말고 충돌과 판단에 필요한 추가 정보를 보고한다.
- 위키에 없는 내용은 코드, 설정, 테스트와 구분해 "위키에 기록되지 않음"으로 표시한다.
- 문서의 날짜가 오래되었으면 현재 코드와 일치하는지 확인이 필요하다고 표시한다.

## 답변 형식

질문이 단순하면 짧게 답하고, 작업 판단에 영향을 주는 내용이면 다음을 상황에 맞게 포함한다.

1. 관련 문서: 파일 링크와 제목
2. 핵심 사실과 제약
3. 적용되는 기존 결정
4. 문서 간 충돌 또는 오래된 내용
5. 현재 작업에 대한 영향
6. 추가로 확인할 사항

문서 내용을 그대로 길게 복사하지 말고 작업과 관련된 부분을 요약한다. 위키가 근거라면 경로를 명시한다.

## 금지 사항

- 위키 파일, .agents/wiki-path, .agents/wiki-structure.md를 수정하지 않는다.
- 읽기 중 새 문서, index.md, logs 디렉토리 또는 로그 파일을 만들지 않는다.
- 위키가 없다고 임의의 기본 구조를 생성하지 않는다.
- 문서에 없는 결정을 사실처럼 추가하지 않는다.

## 예시

~~~text
사용자: $wiki-read 결제 시스템을 변경하기 전에 관련 결정을 찾아 요약해줘
처리:
1. wiki-structure.md와 루트 index.md 읽기
2. decisions/index.md와 관련 하위 index.md 탐색
3. 결제 관련 ADR과 개념 문서 읽기
4. 현재 상태, 적용할 결정과 충돌을 파일 링크와 함께 요약
~~~
